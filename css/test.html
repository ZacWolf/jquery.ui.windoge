The WINDOGE 8 project started as a lark. It originated with a post in r/wallpapers/ on Reddit.com. Someone had created a background in the style of Windows 8™, but with Doge from the Doge meme.

Originally I created a Windows Theme using these backgrounds and some soundbytes from Portal 2, but then I remembered that I had actually started an HTML5/JS/CSS3 Windows Manager for use in my website. I developed it as a jQuery/jQuery-UI addon, and then created an Open Source project for the code hosted on GitHub.
