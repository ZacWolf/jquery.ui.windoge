Jquery.ui.windoge

Add windows to your UI. Windows with maximize/restore minimize capabilities,
and that minimizes into an icon dock.

download .ZIP download .TGZ

jquery.ui.windoge is a JavaScript windows manager for your UI.

Install

bower install zacwolf/jquery.ui.windoge
